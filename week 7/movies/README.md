# Movies

SQL queries to answer questions about a database of movies.

database 

## Usage

Run the following commands in order to Execute this program:

```bash
$ cat filename.sql | sqlite3 movies.db
```

You can also run

```bash
$ cat filename.sql | sqlite3 movies.db > output.txt
```
